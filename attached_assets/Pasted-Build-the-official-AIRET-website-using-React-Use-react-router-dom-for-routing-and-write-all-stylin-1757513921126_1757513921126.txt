Build the official AIRET website using React. Use react-router-dom for routing, and write all styling in pure CSS files (no external UI frameworks, no icon/font libraries). 
The site must follow AIRET’s brand structure, with **sections alternating between White and Black backgrounds**. 
Default text language is English. The Home hero section must use /public/videos/movie.mp4 as a **fullscreen autoplay background video**.

[Routing]
- Home    → "/"
- About   → "/about"
- Product → "/product"
- Patents → "/patents"
- News    → "/news"

[Section Theme Rules]
- First section White → then Black → continue alternating.
- .section--white: background #fff, text #000, muted text dark gray, divider light gray, cards white with thin black borders.
- .section--black: background #000, text #fff, muted text light gray, divider dark gray, cards dark gray.
- SectionHeader must automatically adapt for contrast (white bg → black title, black bg → white title).
- Cards/buttons/links use only brightness/opacity changes (no colors).

[Style Guide]
- variables.css:
  :root{
    --w-bg:#fff; --w-fg:#000; --w-muted:#444; --w-line:#ddd; --w-surface:#fff;
    --b-bg:#000; --b-fg:#fff; --b-muted:#bbb; --b-line:#222; --b-surface:#111;
    --radius:12px; --space:16px; --maxw:1200px;
    --font:system-ui,-apple-system,Segoe UI,Roboto,Helvetica,Arial,sans-serif;
    --shadow:0 2px 12px rgba(0,0,0,.08);
  }
- global.css:
  - Simple reset (*, html, body, img, a, button)
  - body{font-family:var(--font); margin:0;}
  - .container{max-width:var(--maxw); margin:0 auto; padding:calc(var(--space)*2);}
  - .section{padding:clamp(56px,8vw,120px) 0;}
  - .section--white{background:var(--w-bg); color:var(--w-fg);}
  - .section--black{background:var(--b-bg); color:var(--b-fg);}
  - .card{border-radius:var(--radius); border:1px solid; overflow:hidden;}
    .card--white{background:var(--w-surface); border-color:var(--w-line);}
    .card--black{background:var(--b-surface); border-color:var(--b-line);}
  - a,button:hover{filter:brightness(.9);}
  - :focus-visible{outline:2px solid currentColor; outline-offset:2px;}
  - @media (prefers-reduced-motion:reduce){*{animation:none !important; transition:none !important;}}

[Global Layout]
- Navbar: white background + black text, subtle shadow on scroll; active link underlined or thicker.
- Footer: always white theme, thin gray divider, black text, placeholder company info, © AIRET.
- SectionHeader: h2 + subtitle text, adapting to section theme.
